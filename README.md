## ✨ Features

- **Daily Habit Management** — Add, edit, and track unlimited habits with custom goals and frequencies
- **Smart Streaks & Progress** — Automatic streak calculation and motivational visuals
- **AI-Powered Suggestions** — OpenAI API integration for personalized habit advice based on your history
- **On-Device Intelligence** — CoreML model for habit pattern recognition (works offline)
- **Beautiful Visualizations** — Weekly & monthly progress using Swift Charts
- **Smart Reminders** — Local push notifications at custom times using UserNotifications
- **Clean Modern UI** — Dark mode, smooth animations, and Apple design language
- **Planned** — iCloud Sync + Apple Watch companion app

## 🛠 Tech Stack

- **SwiftUI** — Modern declarative UI
- **SwiftData** — Persistent storage & relationships
- **CoreML** — On-device machine learning
- **OpenAI API** — Personalized AI coaching
- **Swift Charts** — Rich data visualizations
- **UserNotifications** — Smart local reminders
- **MVVM + Repository Pattern** — Clean architecture
- **async/await & Concurrency** — Modern Swift
